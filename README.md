# ACU COBA Professional Development Platform

A comprehensive web-based platform designed to support professional development for students at Abilene Christian University's College of Business Administration (COBA).

## Overview

This platform provides COBA students with centralized access to career resources, professional development materials, networking opportunities, and job/internship postings. The goal is to empower students with the tools and connections they need to succeed in their professional careers.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Sugira-rob/coba-professional-development.git
```

2. Navigate to the project directory:
```bash
cd coba-professional-development
```

3. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Alternatively, you can use a local web server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server (requires npm install -g http-server)
http-server
```

Then navigate to `http://localhost:8000` in your browser.

## Usage

### Navigation
- Use the navigation bar at the top to quickly jump to different sections
- Click on "Explore Resources" button to view professional development materials
- Scroll through the page to discover various resources and opportunities

### Viewing Opportunities
- Navigate to the Opportunities section
- Use the tabs to switch between Internships, Full-Time Jobs, and Scholarships
- Click "Apply" buttons to access application forms (demo functionality)

### Accessing Resources
- Browse through different resource categories
- Click on resource buttons to download templates and guides (demo functionality)
